This project focuses on predicting the amount of electricity that households will consume by looking at data from smart meters.
It uses Apache Spark to handle big data efficiently and machine learning models to make predictions about future energy use.
The system helps make energy distribution more effective, cut down on waste, and assist with better planning for homes, utility
providers, and government officials. 

Rising population and widespread use of smart devices are driving a swift surge in electricity demand. Yet, suboptimal energy 
usage and inaccurate predictions can cause grid overloads and result in wasted power. 
In this project, I am trying to: 
•	Examine extensive smart meter datasets that record household energy consumption. 
•	Perform data preprocessing and feature engineering using Spark. 
•	Build regression models to predict future energy consumption. 
•	Evaluate several machine learning methods, such as Linear Regression, Random Forest Regression, and Gradient Boosted Trees.
